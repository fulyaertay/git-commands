
### Mostly used Git commands simplified 👇👇

🌱 Start a new project:
```
git init
git clone
git config
```

➕ Commit changes:
```
git add
git commit
```

🌿Work with branches
git branch
git checkout
git merge

🔄️ Sync changes:
```
git pull
git push
```

💁‍♂️ Helper commands:
```
git log
git status
git diff
```
